Placement

In the paper, placement is part of the layout synthesis stage. The public implementation uses Gurobi to solve a placement problem with area and weighted-HPWL objectives, module variants, rotation choices, halo spacing, and IO placement rules.

Command

python3 perform_placement.py --config EXAMPLES/LNA_1/config.json

Inputs

  • project_name/stage_2/<topcell>_design.json
  • placement.placement_constraints
  • placement.timelimit
  • placement.aspect_ratio
  • placement.level
  • placement.weight_hpwl

Outputs

  • stage_3/placement/<topcell>_placement.json
  • stage_3/placement/<topcell>_placement.gds
  • stage_3/placement/primitives/*.gds

Supported Constraint Types

  • halo
  • keepouts
  • symmetry
  • alignment
  • regions
  • proximity
  • ordering

The shipped examples are minimal, but the code supports much richer constraint files than the simplest examples show.
